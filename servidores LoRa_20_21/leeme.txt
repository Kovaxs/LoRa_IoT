
 <loraserver.py> es el servidor que simula la red lora. Este servidor transmite datos via http.
 
 <middleware.py> es el servidor que recibe los datos de loraserver. (Este es el que teneis que completar con vuestro código).


NOTAS:
	1) Para la ejecución de los servidores, debeis ejecutar cada servidor en una consola independiente. Primero hay que ejecutar el servidor
	   middleware.py y posteriormente ya podemos transmitir mensajes utilizando el servidor loraserver.py

	2) Antes de ejecutar los servidores debeis aseguraros de que las librerías que utilizan las teneis instaladas. (socket y request).
